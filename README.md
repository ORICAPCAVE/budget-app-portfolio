# Budget App Starter

Starter desktop budget application for Java + JavaFX + SQLite.

## Stack
- Java 21
- JavaFX 21
- Gradle
- SQLite
- JUnit 5

## Included
- Gradle build files
- JavaFX app entry point
- Dashboard FXML + controller
- Core models for income, expenses, debts
- Budget summary service
- Debt payoff calculation service
- SQLite schema bootstrap
- Test example for payoff math

## Open in IntelliJ IDEA Ultimate
1. Open IntelliJ IDEA Ultimate.
2. Choose **Open**.
3. Select the `budget-app-starter` folder.
4. Let IntelliJ import the Gradle project.
5. Use the Gradle tool window to run `application > run`.
